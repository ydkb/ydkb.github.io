刷机方式：
首先重点，这不是gh60，不要用zadig去安装驱动，这个刷固件是免驱的。
19.3.8前购买的，拔掉数据线，按住左上角Esc键同时插入数据线，进入刷机模式，此时可放开Esc。再将hex拖到ydkbs-relflash.exe，固件将自动刷新。
固件刷完后，再按一下Esc或重新插USB线，退出刷机模式。
还有疑问参看http://help.ydkb.io内的更详细说明。

19.3.8后购买的，按Esc插线电脑显示一个名为STM60的U盘，网站上下载选择下载UF2格式的，然后不用管文件名，直接拖到U盘里就行。它会自动刷新完后退出U盘

补充说明:
背光设置和RGB灯带设置已经保存，因为更换了keymap的保存地址，之前的hex上传后无法读取正常keymap的，重新设置一下再下载。

Reflash Firmware:
STM60 uses STM32 HID bootloader. So it's no need to install any driver manually for it.
Disconnect usb cable from the keyboard first. Drag .hex file to ydkb-relflash.exe, as it says next you need get your device into bootloader.
Now, keep holding Esc key and insert the usb cable to your keyboard . When it begins flashing, Esc can be released.
When reflash finished, press esc to exit bootloader.
visit http://help.ydkb.io for more details.

More to say:
Backlight mode can't be saved yet.
